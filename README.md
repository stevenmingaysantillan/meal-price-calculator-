Description
A practical, granular financial calculator designed to compute individual and group subtotals for restaurant orders. Unlike a basic meal calculator, this program breaks down costs into distinct categories for both adults and children to handle realistic family dining scenarios.

Key Features
Detailed Bill Breakdown: Tracks individual prices for meals, drinks, and extra add-ons separated by age group.

Mathematical Type Casting: Effectively handles floating-point math (float) for monetary values and integers (int) for party sizes.

Automatic Tax & Change Computation: Dynamically calculates local sales tax, adds it to the subtotal for a grand total, and computes the exact change due based on the client's cash payment.

Clean Formatting: Uses Python's floating-point formatting (:.2f) to strictly display currency output with two decimal places.
